## 📁 Featured Projects

### 🔹 [Farmin AI Assistant](https://github.com/keshava8769/Farmin-AI)
An AI-powered smart farming platform that helps farmers detect crop, poultry, and cattle diseases while maintaining digital farm records.
- **Key Contributions:** Built the application with Streamlit, integrated OpenCV for computer vision, and used Scikit-learn for ML models.
- **Tech Stack:** Python, Streamlit, Scikit-learn, OpenCV, Machine Learning, NumPy

### 🔹 [NutriPal AI](https://github.com/keshava8769/NutriPal-AI)
An intelligent nutrition assistant that provides personalized meal recommendations and food insights, with multilingual support and nutrition PDF analysis.
- **Key Contributions:** Integrated the Google Gemini AI API, implemented prompt engineering for better results, and built a user-friendly Streamlit interface.
- **Tech Stack:** Python, Streamlit, Google Gemini AI, Prompt Engineering, PyMuPDF, Dotenv

### 🔹 [AI Demand Forecasting & Inventory Optimization](https://github.com/keshava8769/Demand-Forecasting)
An AI-powered system that predicts restaurant sales from historical POS data to enable smarter inventory management and reduce food waste.
- **Key Contributions:** Developed a time-series forecasting model using XGBoost, performed comprehensive data analysis with Pandas, and optimized the model for accuracy.
- **Tech Stack:** Python, XGBoost, Time Series Forecasting, Pandas, Scikit-Learn, Data Analytics

### 🔹 [AI-Driven Citizen Grievance & Sentiment Analysis System](https://github.com/keshava8769/Grievance-Analysis)
An NLP-powered system to analyze citizen grievances and perform sentiment analysis, helping authorities prioritize and address public concerns effectively.

### 🔹 [Job Board Portal](https://github.com/keshava8769/Job-Board-portal)
A full-stack web application built with Python Flask, acting as a mini version of LinkedIn/Indeed. It allows recruiters to post jobs and students to search and apply with resume uploads.
- **Key Contributions:** Implemented role-based access control (Recruiter/Student), built search filters, set up resume upload and tracking, and designed the database models.
- **Tech Stack:** Python, Flask, Flask-SQLAlchemy, SQLite/PostgreSQL, HTML5, Bootstrap 5, Jinja2
